# 📊 牛人影音志 (Ikokei Media Log)

> 拒绝一拍脑袋的打分，用严谨的架构记录每一份精神食粮。

牛人影音志是一个专为热爱观影、追番、阅读和游戏的独立玩家打造的私人精神食粮档案馆。 本系统现已全面接入云端数据库，具备强自定义性，支持多维度加权算法，让你每一份打分都有据可查。

**🚀 目前本项目正处于高速迭代与繁荣进化中，更多硬核功能敬请期待！**

---

## ✨ 核心特性

- ☁️ **全量云端漫游**：接入 Supabase，数据与配置双轨同步，多端设备无缝衔接。
- 🚀 **10分制加权算法**：系统内置多套专业评分框架（涵盖动画、Galgame、影视、文学等），支持自定义维度与权重。
- ⚖️ **独创一票机制**：支持“一票赞成（白月光）”与“一票否决（踩雷）”，打破平庸的分数平均值。
- 📈 **个人数据看板**：基于 Chart.js 驱动，支持下钻统计细分项数据，实时呈现领域涉猎饼图与评分正态分布曲线图。
- ✍️ **沉浸式写作 (Zen Mode)**：独创全屏打字机模式，左侧智能悬浮上下文矩阵，右侧提供无干扰创作空间。
- ☰ **极致交互体验**：
  - **原生拖拽引擎**：设置中心全面支持大类与小项的丝滑拖拽排序。
  - **异步防抖锁**：全站核心网络操作配置严格的异步锁定，杜绝因延迟造成的数据重影。
  - **智能重名预警**：输入作品名时自动检索馆藏，防止重复录入。

---

## 🏗️ 架构说明

本项目采用纯粹的无构建 Web 工程化结构组织，以极致轻量运行：
- `index.html` 纯净的语义化骨架，负责界面结构。
- `style.css` 独立的样式表，包含完整的日夜主题适配与流体动效。
- `config.js` 全局静态配置中心，管理状态与框架。
- `app.js` 核心逻辑引擎，直连云端进行数据交互与图表渲染。
