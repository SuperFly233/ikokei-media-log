# 📊 牛人影音志 (Ikokei Media Log)

 拒绝一拍脑袋的打分，用严谨的架构记录每一份精神食粮。

牛人影音志是一个专为热爱观影、追番、阅读和游戏的独立玩家打造的私人精神食粮档案馆。 本系统采用纯本地化存储，具备强自定义性，支持多维度加权算法，让你每一份打分都有据可查。

---

## ✨ 核心特性

- 🚀 10分制加权算法：系统内置多套专业评分框架（涵盖动画、Galgame、影视、文学等），支持自定义维度与权重。
- ⚖️ 独创一票机制：支持“一票赞成（白月光）”与“一票否决（踩雷）”，打破平庸的分数平均值。
- 📈 个人数据看板：基于 Chart.js 驱动，实时演算全站平均分、最常打分数，并自动生成评分分布曲线图。
- ✍️ 沉浸式写作 (Zen Mode)：独创全屏打字机模式，左侧智能悬浮上下文矩阵，右侧提供无干扰创作空间。
- ☰ 极致交互体验：
  - 原生拖拽引擎：设置中心全面支持大类与小项的丝滑拖拽排序。
  - 横向滑动导航：适配不同屏幕宽度，支持滚轮转换与智能点击居中。
  - 智能重名预警：输入作品名时自动检索馆藏，防止重复录入。
- 🔐 数据安全闭环：数据完全存储于浏览器 `localStorage`，支持全量 JSON 存档的导出与一键还原。

---

## 🏗️ 架构说明

本项目采用标准的 Web 工程化结构进行组织：

- `index.html` 纯净的语义化骨架，负责界面结构。
- `style.css` 独立的样式表，包含完整的日夜主题适配与拖拽动效。
- `app.js` 核心逻辑引擎，承载 1100+ 行演算逻辑与图表渲染。

---

## 🛠️ 如何使用

1. 在线访问：通过 GitHub Pages 提供的链接直接打开即可。
2. 离线运行：下载本项目文件夹，双击 `index.html` 即可在任何浏览器中运行，无需服务器环境。
